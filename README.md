# Sr Bracat & o Pósitron

Se você já estudou mecânica quântica alguma vez, tenho certeza que já se deparou com um poço de potencial. Superposição de estados também é uma figurinha carimbada quando falamos dessa área da física. E o que dizer sobre evolução temporal e a equação de Schrodinger? Esses assuntos são alguns dos grandes responsáveis por atribuir à mecânica quântica o seu adjetivo mais utilizado: "contra intuitiva". Claro que o termo surge da total incompatibilidade entre o que é observado no dia a dia e o que se estuda sobre o mundo microscópico. Mas isso não significa que é impossível entender alguma coisa além de entes matemáticos abstratos. É possível visualizar o que acontece e com isso criar intimidade com o universo das partículas minúsculas, de forma a, quem sabe, construir uma intuição sobre elas.

Para isso, o sr Bracat, um gato físico, veio te ajudar! Por meio desse simulador, ele te guiará através de algumas simulações interativas do poço de potencial infinito, explicando cada passo e indicando alguns insights importantes. Para utilizar o simulador, sugiro que você baixe o repositório completo, não apenas o código, já que contém toda a estrutura de arquivos a partir da qual o código foi construído. O código está escrito em python e utiliza as seguintes bibliotecas: numpy, matplotlib e glob. Basta rodar o arquivo simulador.py e seguir as instruções que aparecerão no terminal. Mas cuidado! Alguns avisos chatinhos podem aparecer logo no começo da execução, então ignore o que aparece no começo e leia a partir da reta pontilhada horizontal (-----------------------------------)
